#Installation
***
###Arch Linux
#####install packages form AUR :
```bash
# yaourt grub2-theme-vimix
```
#####Edit /etc/default/grub :
```bash
GRUB_THEME="/boot/grub/themes/Vimix/theme.txt"
```
#####Update grub :
```bash
# grub-mkconfig -o /boot/grub/grub.cfg
```
